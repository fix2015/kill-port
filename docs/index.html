Getting Started

Overview

kill-port aggregates port management from multiple GitHub sources into a searchable registry. Install any management with one command — no cloning repos or copying files.

Quick Start

# Install a management
npx kill-port install <name>

# Install for Cursor
npx kill-port install <name> --tool cursor

# Install globally
npx kill-port install <name> --global

# Search
npx kill-port search react

# List all
npx kill-port list

Commands

Install

kill-port install <name>                    # Install for Claude Code (default)
kill-port install <name> --tool cursor       # Install for Cursor
kill-port install <name> --tool codex        # Install for Codex
kill-port install <name> --global             # Install globally
kill-port install <name> -o ./path           # Custom output path

Search

kill-port search react                       # Search by keyword
kill-port search "api design" --tag backend  # Search with tag filter
kill-port search react --json                # JSON output

List

kill-port list                                # List all
kill-port list --tag testing                  # Filter by tag
kill-port list --tool cursor                  # Filter by tool
kill-port list --json                         # JSON output

Info

kill-port info <name>                         # Detailed info
kill-port info <name> --json                 # JSON output

Tags & Sources

kill-port tags                                # Show all tags with counts
kill-port sources                             # Show all sources

Sources

Skills are aggregated from multiple GitHub repositories maintained by the community.

Update Registry

kill-port update                              # Fetch latest from GitHub
GITHUB_TOKEN=ghp_xxx kill-port update         # With auth for higher rate limits

JSON Output

kill-port list --json
kill-port search react --json
kill-port info <name> --json

All commands support --json for machine-readable output.

Programmatic API

const { findSkill, searchSkills, installSkill } = require('kill-port');

const results = searchSkills('react', { tag: 'frontend' });
const skill = findSkill('react-expert');
await installSkill(skill, { tool: 'claude-code', global: true });
